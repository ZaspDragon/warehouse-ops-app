# WarehouseOS Starter App

A GitHub-ready static warehouse dashboard app.

## What it does right now
- Dashboard with warehouse KPIs
- Operations task board
- Inventory table
- Receiving overview
- Labor and leaderboard views
- Quality / safety tracker
- Maintenance KPI section
- Report cards
- Supervisor notes saved in localStorage
- Search across tasks, inventory, and labor

## How to use on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Make sure `index.html` stays in the root.
4. In GitHub repo settings, turn on **Pages**.
5. Deploy from the `main` branch root.

## Files
- `index.html`
- `styles.css`
- `app.js`
